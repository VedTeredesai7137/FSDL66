Here's a simple **Node.js/Express API** for managing a bookstore. It includes **GET** and **POST** endpoints to manage a list of books (with `title` and `author`). ### Simple REST API for Bookstore 1. **Create a new directory for the project** and initialize it with npm: ```bash mkdir bookstore-api cd bookstore-api npm init -y ``` 2. **Install Express**: ```bash npm install express ``` 3. **Create the API file** (`index.js`): ```javascript // index.js const express = require('express'); const app = express(); const port = 3000; // Middleware to parse JSON bodies app.use(express.json()); // Sample list of books let books = [ { id: 1, title: 'The Great Gatsby', author: 'F. Scott Fitzgerald' }, { id: 2, title: '1984', author: 'George Orwell' }, { id: 3, title: 'To Kill a Mockingbird', author: 'Harper Lee' }, ]; // GET endpoint to fetch all books app.get('/api/books', (req, res) => { res.json(books); }); // POST endpoint to add a new book app.post('/api/books', (req, res) => { const { title, author } = req.body; // Simple validation if (!title || !author) { return res.status(400).json({ message: 'Title and author are required.' }); } // Add the new book to the list const newBook = { id: books.length + 1, // simple auto-increment ID title, author }; books.push(newBook); res.status(201).json(newBook); }); // Start the server app.listen(port, () => { console.log(`Server running at http://localhost:${port}`); }); ``` ### **Explanation**: * **GET `/api/books`**: This endpoint returns a list of all books in the `books` array. * **POST `/api/books`**: This endpoint allows the user to add a new book by sending a JSON object with the `title` and `author` properties. If a book is successfully added, it will return the added book with a generated `id`. ### **How to Run**: 1. Create the project folder and add the above `index.js` code. 2. Run the server: ```bash node index.js ``` This will start the server at `http://localhost:3000`. 3. **Test with Postman**: * **GET Request**: To fetch all books, make a `GET` request to `http://localhost:3000/api/books`. * **POST Request**: To add a new book, make a `POST` request to `http://localhost:3000/api/books` with a JSON body like: ```json { "title": "New Book Title", "author": "New Author" } ``` ### **Example Response**: * **GET** `/api/books`: ```json [ { "id": 1, "title": "The Great Gatsby", "author": "F. Scott Fitzgerald" }, { "id": 2, "title": "1984", "author": "George Orwell" }, { "id": 3, "title": "To Kill a Mockingbird", "author": "Harper Lee" } ] ``` * **POST** `/api/books`: ```json { "id": 4, "title": "New Book Title", "author": "New Author" } ``` This API provides a basic way to manage a list of books using **GET** and **POST** requests.
